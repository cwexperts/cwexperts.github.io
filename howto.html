How to Play

The general idea you need to get is that we start a game on one world and after 10 minutes, we start a different game on another world with our losers while the original game is still going on. By doing this, we can do games in ~13 minutes instead of 23 minutes full games and receive about 9-10 tickets per hour.

Basics

  1. Once you are given access, join #cwexperts1
  2. Use the .w command so our bot can tell you the current world
