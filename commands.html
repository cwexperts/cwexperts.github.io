Main Game Commands

These are the main commands for playing games in #cwexperts

  • .w is used when you want to find out which world to go to
  • .ls is usually only used when you miss joining the game and need people on sara to lobby
  • .lz is used when you need the losers on zammy to lobby for more to join the game
  • .ul is used only if you are the leader or if you were last person to join the game
  • .hw WORLD should be only used by the leader to notify everyone of the next world
  • .mhl should be used if you have to notify everyone of something, such as crashers
  • .lead should be used to pick a random person to be the leader
  • .lead NAME can only be used by halfops+ to force set a leader
  • .salt should be used to pick a random person to be the salt
  • .salt NAME can only be used by halfops+ to force set a salt
  • .zalt should be used to pick a random person to be the zalt
  • .zalt NAME can only be used by halfops+ to force set a zalt
  • .et should be used to pick 3 random people to help lead & s/zalt with ET
  • .et1 NAME can only be used by halfops+ to force set 1st person for ET
  • .et2 NAME can only be used by halfops+ to force set 2nd person for ET
  • .et3 NAME can only be used by halfops+ to force set 3rd person for ET
  • .tick should be to notify the z/salts to leave the game and join the other side
  • .roles can be used to display the people with roles

Database Commands

These are the database commands, mainly for main and alt registrations.

  • .addalt is used when you want to add a new alt and your main is your current IRC nick
  • .editalt is used to edit the alt name if you are using a different alt or changed the name of an alt
  • .main is used in combination with the alt name to find out who the alt owner is
  • .alt is used in combination with the main name to find out who their alt is
  • .idiot can only be used by halfops+ to add a person to the idiot list
  • .genius can only be used by halfops+ to remove a person from the idiot list
  • .idiots can be used by halfops+ to list the current idiots list
  • .warn can only be used by halfops+ to add a warn to a certain player
  • .warns can only be used by halfops+ to show the warns for a player

Other Game-related Commands

These are other game-related commands that are not commonly used.

  • .c is used to check the number of players
  • .nogames is used when games are over in the channel to clear the worlds registered
  • .pentoggle is used to add the person to a list that gets a reminder to exit the pen
  • .penidiots can only be used by halfops+ and is similar to pentoggle, but includes the list of idiots
  • .ticksleft can be used to check the amount of ticks left on the current game
  • .hl can be used to check if your highlight options work
  • .rules links to the rules! plz follow them
  • .howto links to the how to play guide
  • .advert the official cwexperts ad
